# Build and Deploy a Threads App with Real-Time Chat Functionality

![Demo App](https://i.ibb.co/BnGdh10/Group-62.png)





### Setup .env file

```js
PORT=...
MONGO_URI=...
JWT_SECRET=...
CLOUDINARY_CLOUD_NAME=...
CLOUDINARY_API_KEY=...
CLOUDINARY_API_SECRET=...
```

### Build the app

```shell
npm run build
```

### Start the app

```shell
npm start
```
